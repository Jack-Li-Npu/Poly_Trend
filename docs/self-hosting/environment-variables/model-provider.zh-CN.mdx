# 模型提供商环境变量

本文档描述了配置 AI 模型提供商的环境变量。

## Anthropic

### `ANTHROPIC_API_KEY`

- **类型：** 必需
- **描述：** 从 Anthropic 控制台获取的 API 密钥。用于验证对 Claude 模型的请求。
- **示例：** `sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### `ANTHROPIC_MODEL_LIST`

- **类型：** 可选
- **描述：** 控制模型选择中可用的 Claude 模型。使用 `+` 前缀添加模型，使用 `-` 前缀隐藏模型。使用 `-all` 在添加特定模型之前隐藏所有默认模型。
- **默认值：** 所有可用的 Claude 模型
- **示例：** `-all,+claude-opus-4-5-20251101,+claude-sonnet-4-20250514=Claude Sonnet 4`

### `ANTHROPIC_PROXY_URL`

- **类型：** 可选
- **描述：** Anthropic API 请求的自定义代理 URL。适用于通过代理服务器路由或使用兼容的 API 端点。
- **默认值：** `https://api.anthropic.com`
- **示例：** `https://my-proxy.example.com/anthropic`

## Google (Gemini)

### `GEMINI_API_KEY`

- **类型：** 必需
- **描述：** 从 Google AI Studio 获取的 API 密钥。访问 Gemini 模型必需。
- **示例：** `AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### `GEMINI_MODEL_LIST`

- **类型：** 可选
- **描述：** 控制模型选择中可用的 Gemini 模型。
- **默认值：** 所有可用的 Gemini 模型
- **示例：** `-all,+gemini-2.0-flash,+gemini-1.5-pro`

## OpenAI

### `OPENAI_API_KEY`

- **类型：** 必需
- **描述：** 从 OpenAI 平台控制面板获取的 API 密钥。
- **示例：** `sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### `OPENAI_MODEL_LIST`

- **类型：** 可选
- **描述：** 控制可用的 OpenAI 模型。
- **默认值：** 所有可用的 GPT 模型
- **示例：** `-all,+gpt-4o,+gpt-4o-mini`

### `OPENAI_PROXY_URL`

- **类型：** 可选
- **描述：** OpenAI API 请求的自定义代理 URL。
- **默认值：** `https://api.openai.com/v1`
- **示例：** `https://my-proxy.example.com/openai/v1`

## 模型列表配置

`*_MODEL_LIST` 变量支持以下语法：

| 语法 | 描述 |
|------|------|
| `model-id` | 使用默认名称添加模型 |
| `+model-id` | 显式添加模型 |
| `-model-id` | 隐藏特定模型 |
| `-all` | 隐藏所有默认模型 |
| `model-id=显示名称` | 使用自定义显示名称添加模型 |

### 示例

```bash
# 仅显示特定模型
ANTHROPIC_MODEL_LIST="-all,+claude-opus-4-5-20251101,+claude-sonnet-4-20250514"

# 隐藏特定模型
ANTHROPIC_MODEL_LIST="-claude-3-5-haiku-20241022"

# 添加自定义显示名称
ANTHROPIC_MODEL_LIST="+claude-opus-4-5-20251101=Claude Opus 4.5 (最新)"
```
